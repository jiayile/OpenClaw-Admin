# Cron 可视化编辑器需求文档

## 功能描述

为 OpenClaw Admin 提供可视化的 Cron 定时任务编辑器，降低 Cron 表达式配置门槛，支持实时预览、模板选择和执行历史查看。

## 用户故事

### 用户故事 1：可视化配置 Cron 表达式
**作为** 系统管理员  
**我想要** 通过可视化界面配置定时任务，而不是手动编写 Cron 表达式  
**以便** 减少配置错误，提高工作效率

**验收标准：**
- [ ] 提供分钟、小时、日期、月份、星期的可视化选择器
- [ ] 实时显示生成的 Cron 表达式
- [ ] 支持常用模板（每日、每周、每月、每小时等）
- [ ] 提供 Cron 表达式语法提示和校验
- [ ] 支持保存和加载已配置的任务模板

### 用户故事 2：查看任务执行历史
**作为** 系统管理员  
**我想要** 查看定时任务的执行历史记录  
**以便** 监控任务运行状态，排查问题

**验收标准：**
- [ ] 显示任务执行时间、状态、输出结果
- [ ] 支持按任务名称、时间范围筛选
- [ ] 支持查看单次执行的详细日志
- [ ] 执行失败时显示错误信息

### 用户故事 3：快速创建常用任务
**作为** 系统管理员  
**我想要** 使用内置的任务模板快速创建常用任务  
**以便** 减少重复配置工作

**验收标准：**
- [ ] 提供内置模板：晨报、健康巡检、数据备份、日志清理等
- [ ] 模板支持自定义参数
- [ ] 一键应用模板创建任务

## 技术依赖关系

### 前端依赖
- **Vue 3** - 基础框架
- **Naive UI** - UI 组件库
- **cron-parser** - Cron 表达式解析和验证
- **dayjs** - 时间处理

### 后端依赖
- **node-cron** - 任务调度引擎
- **SQLite** - 任务执行历史存储
- **现有 API 层** - 任务 CRUD 接口

### 技术栈
```
前端: Vue 3 + Naive UI + TypeScript
后端: Node.js + Express + node-cron + SQLite
```

## 功能模块设计

### 1. Cron 表达式编辑器组件
```
- 可视化选择器（5 个维度）
- 实时表达式预览
- 语法校验和错误提示
- 常用频率快捷按钮
```

### 2. 任务模板库
```
- 内置模板列表
- 模板参数配置
- 自定义模板保存
```

### 3. 执行历史查看器
```
- 历史列表（分页）
- 筛选器（任务名、时间、状态）
- 详情弹窗（日志、输出）
```

### 4. API 接口
```
GET    /api/cron/templates       # 获取模板列表
POST   /api/cron/tasks           # 创建任务
GET    /api/cron/tasks           # 获取任务列表
GET    /api/cron/tasks/:id       # 获取任务详情
PUT    /api/cron/tasks/:id       # 更新任务
DELETE /api/cron/tasks/:id       # 删除任务
GET    /api/cron/tasks/:id/history # 获取执行历史
```
